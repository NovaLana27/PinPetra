# PinPetra

PinPetra is a polished, mobile-first concept app for discovering, saving, and organizing fashion and lifestyle inspiration. Inspired by a Pinterest-style experience, it combines elegant visuals, a soft blush-toned palette, and immersive browsing flows for users who love curating ideas for style, beauty, home décor, and creative living.

## ✨ What this project includes

- A complete interactive prototype flow from splash screen to onboarding, login, interests, feed, search, and profile
- A Pinterest-inspired masonry feed with categories such as bags, shoes, looters, clothes, braids, makeup, flowers, DIY, accessories, and home décor
- Save-to-board interactions, notifications, profile settings, and a modern mobile UI
- A feminine, minimal, editorial design language built around blush pink, dusty rose, berry pink, and ivory tones

## 🛠️ Tech stack

- React 18
- Vite
- Tailwind CSS
- Lucide React

## ▶️ Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser, typically http://localhost:5173.

## 📝 Notes

- This is a frontend prototype with mock, in-memory data. Refreshing the page resets saved items and boards.
- Images are currently sourced from placeholder services for visual prototyping.
- The experience is designed to be responsive and feel native on mobile screens.
